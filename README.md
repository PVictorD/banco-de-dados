# E-commerce de Peças de PC – Scripts SQL

Este repositório contém todos os scripts SQL utilizados no projeto de modelagem do banco de dados do e-commerce.

## Conteúdo
- **01_create_tables.sql** – Criação das tabelas (DDL)
- **02_insert_data.sql** – Inserção de dados (INSERT)
- **03_select_queries.sql** – Consultas (SELECT)
- **04_updates.sql** – Atualizações (UPDATE)
- **05_deletes.sql** – Exclusões (DELETE)

## Como executar
1. Importe todos os arquivos para MySQL Workbench ou PGAdmin.
2. Execute primeiro `01_create_tables.sql`.
3. Execute `02_insert_data.sql` para povoar o banco.
4. Utilize os demais scripts conforme necessidade.

## Tecnologias
- SQL (MySQL ou PostgreSQL)
- Modelagem Relacional
- Normalização 1FN, 2FN e 3FN
